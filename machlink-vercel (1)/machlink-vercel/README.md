# MachLink China — Used Milling Machines

B2B lead generation website for used milling machines from China.

## Deploy to Vercel (Step by Step)

1. Upload this folder to a GitHub repository
2. Go to https://vercel.com → Sign in with GitHub
3. Click "Add New Project" → Import your GitHub repo
4. Framework Preset: **Other** (no framework)
5. Root Directory: leave as `/` (default)
6. Click **Deploy**

Done. Vercel will give you a URL like `https://machlink-china.vercel.app`

## Pages

- `/` → Home page (index.html)
- `/products` → Product listing
- `/product-detail` → Product detail
- `/about` → About us
- `/contact` → Contact + Formspree form
